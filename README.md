# 🏥 HealCare - Smart Stunting Monitoring

HealCare adalah platform digital terpadu untuk memantau tumbuh kembang anak guna mencegah stunting secara dini. Didesain dengan antarmuka modern dan sistem keamanan yang kuat.

## ✨ Fitur Utama
* **Dashboard Real-time**: Visualisasi data pertumbuhan anak yang intuitif.
* **Manajemen Data Anak**: Pencatatan tinggi badan, berat badan, dan lingkar kepala.
* **Multi-role Access**: Akses khusus untuk Admin, Petugas Kesehatan, dan Orang Tua.
* **Glassmorphism UI**: Tampilan modern dengan efek transparansi dan blur.

## 🚀 Tech Stack
* **Framework**: Laravel 11
* **Styling**: Tailwind CSS
* **Interactivity**: Alpine.js
* **Database**: MySQL
